# 📰 Fake News Detection Using Machine Learning

A Machine Learning and NLP project that classifies news articles as **Fake** or **Real** using **TF-IDF Vectorization** and **Logistic Regression**. The model is trained on labeled news datasets and can predict the authenticity of unseen news articles with high accuracy.

## 🚀 Features
- Fake vs Real News Classification
- Text Preprocessing
- TF-IDF Feature Extraction
- Logistic Regression Model
- High Prediction Accuracy

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK

## 📊 Workflow
Dataset → Text Preprocessing → TF-IDF → Logistic Regression → Prediction

## 🎯 Objective
To combat misinformation by automatically detecting fake news using Machine Learning and Natural Language Processing techniques.
